// RESEARCH
Paper & Contributions

Our IEEE-format research paper documents the full NightRestore system, ablation study, and deployment.

NightRestore: A NAFNet-Based Universal Night Image Restoration System with Adaptive Multi-Degradation Pipeline
Prakhar Sharma · Atharva Jaind · Avanti Kate · Prajwal Deshpande · Niraj Chavan
D Y Patil International University, Pune · Dept. Computer Applications · 2025
We present NightRestore, a unified image restoration system built on NAFNet that extends the original activation-free architecture with multi-task training on LOL-v2 and Flare7K, automatic degradation detection, Dark Channel Prior defogging, sliding window inference with Hanning blending, and adaptive post-processing. Our final model achieves PSNR of 21.31 dB and SSIM of 0.823 on the LOL-v2 validation set — a significant improvement over the self-supervised baseline of 13.30 dB.

// CONTRIBUTIONS
7 Novel Contributions
01 / ARCHITECTURE
DirectNAFNet
Reimplemented NAFNet with width=32, encoder blocks [2,2,2,4], 10.6M params, skip-init β,γ=10⁻³. Matches original 65 GMACs.
02 / AUGMENTATION
NightSynthesizer
Stochastic pipeline applying Gaussian noise σ~U(5,50), fog via atmospheric model, rain streaks, and Flare7K overlays.
03 / RUNTIME
Automatic Degradation Detector
Statistical classifier using mean luminance μ_L, contrast σ_L, and Laplacian variance λ to activate correct pre-processing.
04 / PRE-PROCESSING
DCP Defogging Stage
Dark Channel Prior with ω=0.85, t_min=0.15. Decouples haze removal from the neural network for foggy inputs.
05 / INFERENCE
Hanning Window Blending
256px patches, 64px overlap, weighted by w(i,j)=hann(i)·hann(j). Supports arbitrary resolution inputs seamlessly.
06 / POST-PROCESSING
Adaptive Post-Pipeline
Bilateral denoising → LAB color correction → adaptive CLAHE → conservative brightness scaling → conditional unsharp mask.
07 / EVALUATION
Ablation Study (v2→v13)
13 training versions tracked from 13.30 dB (self-supervised) → 15.15 dB (MoE) → 19.05 dB → 21.31 dB (final, +60%).
BASELINE
NAFNet (Chen et al., 2022)
Original paper: 40.30 dB PSNR on SIDD denoising, 33.69 dB on GoPro deblurring. SimpleGate replaces all nonlinear activations.
