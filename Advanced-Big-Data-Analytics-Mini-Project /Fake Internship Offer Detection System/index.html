Syllabus Alignment

MOD I
Introduction to Big Data Analytics — 10 V's of Big Data, analytics lifecycle, types of analytics (descriptive → cognitive), data quality challenges. Covered in dataset exploration and project framing.
MOD II
Statistical Foundations — PDF/CDF (Lab 01), Binomial/Normal distributions (Lab 02), t-test, z-test, chi-square, ANOVA (Lab 03). All implemented in Cell 5E of notebook.
MOD III
Big Data Processing Frameworks — Apache Spark RDDs (Lab 07), MapReduce (Lab 08), DAG execution, PySpark session. Cells 5C and 5D in notebook.
MOD IV
Analytics Techniques — Feature engineering (17 features), Logistic Regression, Random Forest, K-Means clustering (Lab 06), Association Rules using MLlib.
MOD V
Tools & Streaming — Kafka real-time streaming simulation, SparkSQL (Lab 04), Plotly dashboards, real-world case study in student fraud protection.

▶ How to Run Everything — Step by Step

Follow these steps in order to run the full project from scratch.
PART 1 — Run the Colab Notebook
1
Open Google ColabGo to colab.research.google.com → File → Upload Notebook → select FakeInternship_OPTIMIZED.ipynb
2
Enable GPU (Optional but faster)Runtime → Change runtime type → Hardware accelerator → T4 GPU → Save
3
Run Cell 1 (Install — ~2 min first time)Click ▶ on Cell 1. It installs Java 17 + PySpark + all packages. Only slow the first time — subsequent runs use cache.
4
Run Cell 2 (Imports)Click ▶ on Cell 2. All Python libraries get imported. Should complete in <10 seconds.
5
Run Cell 3 — Load Dataset (2 options) Option A: Upload your kaggle.json when prompted to download the real 17,880-record EMSCAD dataset.
Option B: Press Cancel — synthetic fallback dataset loads automatically. Both work fine.
6
Run Cells 4A → 4H (Data Preprocessing)These cells do EDA, missing value analysis, class imbalance check, outlier detection. Run them in order by clicking each ▶.
7
Run Cells 5A → 5F (Feature Eng + Spark + Stats)5A = Lexicon · 5B = 17 features · 5C = PySpark RDDs · 5D = MapReduce · 5E = Hypothesis Tests · 5F = Kafka Stream. Run each ▶ in order.
8
Run Cell 6 (Train ML Models)Trains Logistic Regression, Random Forest, K-Means, Association Rules using PySpark MLlib. Takes ~1–2 min.
9
Run Cells 7–8 (Results + Detection)Cell 7 = Plotly validation dashboard · Cell 8A/8B = built-in test samples · Cell 8C = paste your own JD · Cell 8D = upload a PDF/DOCX file
10
Run Final Cell (Summary)Prints the complete project summary and stops Spark session cleanly.
PART 2 — Run this Frontend Website
A
Download index.htmlSave the index.html file to any folder on your PC, e.g. C:/FakeGuard/index.html
B
Open folder in VS CodeOpen VS Code → File → Open Folder → select the folder containing index.html
C
Install Live Server extensionPress Ctrl+Shift+X → search Live Server by Ritwick Dey → Install
D
Start Live ServerRight-click index.html → "Open with Live Server" OR click Go Live in the bottom-right status bar. Opens at http://127.0.0.1:5500
E
Use the DetectorGo to Detector tab → Upload any PDF/DOCX/TXT job offer file → click Analyze. Results appear instantly — no internet needed.
